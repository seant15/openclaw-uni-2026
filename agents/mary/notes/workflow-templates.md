# Mary Agent Notes

## Daily Digest Template

Mary generates daily digests with the following structure:

```
🌸 Good Morning Sean — Here's your daily digest

## 📋 Yesterday's Wins
- [Completed items from memory/todos]

## 🎯 Today's Focus
- [Priority items for today]

## 📚 New Knowledge
- [Any new info/articles/notes added]

## 📎 Documents Pending Review
- [Docs waiting for Sean's attention]

## 💭 Gentle Reminder
- [Soft nudge on overdue items or self-care]
```

## Document Organization Schema

```
/workspace/docs/
├── inbox/          # New docs awaiting processing
├── active/         # Currently relevant docs
├── archive/        # Completed/older docs
├── notes/          # Raw notes and thoughts
└── templates/      # Reusable templates
```

## Knowledge Capture Workflow

1. **Inbox** — New info goes to inbox first
2. **Process** — Mary reviews, tags, summarizes
3. **Organize** — File into appropriate category
4. **Connect** — Link related items
5. **Surface** — Bring relevant knowledge to Sean's attention

## Search & Research Format

When helping Sean research:
1. Clarify the question
2. Search and gather sources
3. Summarize findings with citations
4. Present options with pros/cons
5. Offer recommendation (if appropriate)
