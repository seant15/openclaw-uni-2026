# Mary's Weekly Checklist

## Pre-Task: Check Current Status

Before starting, check the database status:
- [ ] Count total "待处理" items
- [ ] Count items added this week
- [ ] Identify any high-priority items

---

## Main Task: Organize Pending Items

For each item with Status = "待处理":

### Step 1: Review Content
- [ ] Open the Link and review the content
- [ ] Understand the main topic and value
- [ ] Determine if it's LEARNING or SWIPE

### Step 2: Fill Metadata

#### Content Type
- [ ] Set to `LEARNING` if educational/how-to
- [ ] Set to `SWIPE` if reference/example/inspiration

#### Tags (Select all that apply)
- [ ] `google ads` - Google Ads platform content
- [ ] `Facebook Ads` - Meta/Facebook Ads content
- [ ] `ai` - AI tools and applications
- [ ] `video` - Video marketing/content
- [ ] `email` - Email marketing
- [ ] `SEO` - Search engine optimization
- [ ] `CRO` - Conversion rate optimization
- [ ] `Copy` - Copywriting
- [ ] `design` - Design/UX
- [ ] `tracking` - Analytics and tracking
- [ ] `tiktok ads` - TikTok advertising
- [ ] `Pinterest` - Pinterest marketing
- [ ] `Linkedin Ads` - LinkedIn advertising

#### Industry (Select all that apply)
- [ ] `ECOM` - E-commerce businesses
- [ ] `LeadGen` - Lead generation businesses
- [ ] `B2B` - B2B businesses
- [ ] `AI` - AI industry
- [ ] `ALL` - Applicable to all
- [ ] `NFT` - NFT/Web3
- [ ] `Other` - Other industries

#### Original Source
- [ ] `X` - X/Twitter
- [ ] `Youtube` - YouTube
- [ ] `Blog/Article` - Blog or article
- [ ] `TikTok` - TikTok
- [ ] `Pdf` - PDF document
- [ ] `Web` - General web
- [ ] `email` - Email newsletter
- [ ] `Facebook` - Facebook

### Step 3: Extract Key Insights

Write 2-3 sentences summarizing:
- [ ] Core value of the content
- [ ] Key takeaway or lesson
- [ ] When/where it applies

**Example formats**:
```
Google's new Performance Max audience signals can improve dental practice 
conversions by 15-20%. Best applied to practices with clear service categories 
and sufficient conversion data.
```

```
Facebook creative best practices emphasize 3-second hooks and 9:16 vertical 
format for mobile. Particularly effective for awareness campaigns targeting 
25-45 age demographic.
```

### Step 4: Set Priority

- [ ] `🔴 高` - Directly relevant to current active projects
- [ ] `🟡 中` - Relevant to business but not urgent
- [ ] `🟢 低` - General learning/reference

**Priority Guidelines**:
- 🔴 High: Content about Google Ads when we have active Google Ads clients
- 🟡 Medium: Content about new platforms we're considering
- 🟢 Low: General industry trends not immediately applicable

### Step 5: Update Status

- [ ] Change Status from "待处理" to "已整理"

## Monthly Goal Check (First Monday of Month)

### Review Previous Month
- [ ] Total items collected: [N]/30
- [ ] Organization rate: [N]%
- [ ] High priority digested: [N]/6
- [ ] Items applied: [N]/5

### Adjust Goals if Needed
- [ ] If collection < 30 → increase saving frequency
- [ ] If digestion < 20% → reduce collection or increase reading time
- [ ] If application < 5 → improve Action Items extraction
- [ ] If backlog > 7 days → increase organization frequency

---

## Tips for Efficiency

### Batch Processing
- Group similar content types together
- Process all Google Ads content, then all Facebook content, etc.

### Quick Triage
- If content is clearly low value → mark as "已归档"
- If content is duplicate → check if already exists

### Use Templates
- Keep common Key Insights templates for similar content
- Copy/paste and modify rather than writing from scratch

### Time Management
- Aim to process 10-15 items per hour
- If backlog > 50 items, consider splitting across multiple sessions
